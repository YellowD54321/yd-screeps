- room memory（我忘記這能不能新增屬性了，需要確認一下）新增「常走的路徑」屬性，紀錄 RCL <= 2 時 creeps 走過的路。每次 creeps 行動前都會將當下 x y 資料記錄在常走路徑欄位內，如果該座標已經有資料則跳過。
